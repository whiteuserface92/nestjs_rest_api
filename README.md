# WEBAPI

## Description
- Admin-ui, NativeApp(IOS/Android) 에서 사용되는 API
## Installation

```bash
$ npm install
$ npm i -g @nestjs/cli
```

## Running the app


```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
